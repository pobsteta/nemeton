# Naturalness & Wilderness Character Indicators (Family N)

Functions for calculating naturalness and wilderness indicators: - N1:
Infrastructure distance (remoteness from human influence) - N2: Forest
continuity (continuous patch size) - N3: Composite naturalness index
(integrating multiple dimensions)
