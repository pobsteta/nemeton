# Set language manually

Set language manually

## Usage

``` r
nemeton_set_language(lang = c("fr", "en"))
```

## Arguments

- lang:

  Language code ("fr" or "en")

## Examples

``` r
if (FALSE) { # \dontrun{
# Set French
nemeton_set_language("fr")

# Set English
nemeton_set_language("en")
} # }
```
